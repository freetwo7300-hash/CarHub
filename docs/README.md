## 🚀 Quick Start

```bash
# Clone and install
git clone https://github.com/Mostafa-SAID7/CarHub.git
cd CarHub
pnpm install

# Start development
pnpm dev

# Open http://localhost:3000
```

## ❓ FAQ

**Q: What Node.js version is required?**
A: Node.js 18.17+ or 20+

**Q: Can I use npm instead of pnpm?**
A: We recommend pnpm for faster installations, but npm should also work.

**Q: How do I deploy to production?**
A: See [Deployment Guide](./DEPLOYMENT.md) for comprehensive deployment instructions.

**Q: Can I contribute?**
A: Yes! See [Contributing Guide](./CONTRIBUTING.md) for guidelines.
